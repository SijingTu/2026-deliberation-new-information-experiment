The model 1 inside the archive computes the stationary distribution based on the assumption that every time, two points are uniformly randomly draw from $[0, 1]$. 
This assumption is not accurate once we assume there a $n$ points: only finite number of points in $[0, 1]$ will be sampled. 
In addition, we consider the domain within $\{1, 2, \ldots, n\}$.
